# SID

VICE allows the SID to be mapped to $8F00 or $E900.

ORB's No Pets Allowed demo supports the SID at $8F00.

Both CBM-II B and P-series machines have a 6581 SID chip

CBM-II B-series has a 6581 SID at $DA00-$DB00 (overclocked at 2 MHz, so registers cannot be read).
